HA Cluster Demo

This is a demonstration of the solace-ha clustering library which provides stateful message-driven applications with fault tolerance. This library handles all the state transitions from startup, to backup, to active, etc. providing applications with a simple event-driven interface for all important events.

To demonstrate the capabilities enabled by this library, I've created a sample Mock Order Matching Engine that can be deployed in fault tolerant clusters of instances. These instances use this HA library to:

  1. Signal to the instances when they are acting as Primary or Backup;
  2. Synchronize their state with the other instances at all times.
